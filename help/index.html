Mac Mouse Fix

Notarization Issues

If you're using macOS 10.15 Catalina, you might have encountered the following error message when trying to install Mac Mouse Fix.

Why does this issue occur?

There is a new security feature in Catalina called Notarization. Notarized apps are validated by Apple, so you can be sure they don't contain any malware. Notarization is a great safety measure in principle, but there are some issues in the way it is currently implemented which might prevent you from using unnotarized Preference Panes entirely. Notarizing an app costs 100 dollars per year, which unfortunately makes it a nonsolution for free and open-source software like Mac Mouse Fix.
Apple will likely fix these issues in the future, but for now, you will, unfortunately, have to rely on a workaround to get Mac Mouse Fix to run on Catalina.

The Solution

Morgan Newman, who has been wonderfully supportive of this little project, figured out how to install Mac Mouse Fix despite these issues. He found that, in order to run Mac Mouse Fix, you have to disable Gatekeeper – a security feature in macOS which prevents the execution of apps that are not signed and notarized with Apple.

First Solution – Disabling Gatekeeper entirely

Disabling Gatekeeper will make it easier for you to run malicious software on accident.

  1. Open Terminal and enter the following command

    2. sudo spctl --master-disable

  2. If System Preferences is running – close it

Second Solution – Adding a Gatekeeper exception for Mac Mouse Fix

Adding an exception does not compromise the security of your system.

  1. Install Mac Mouse Fix
  2. Open Terminal and enter the following command

    3. sudo spctl --add ~/Library/PreferencePanes/Mouse\ Fix.prefPane; sudo spctl --add /Library/PreferencePanes/Mouse\ Fix.prefPane

  3. Uninstall Mac Mouse Fix
    • Go to System Preferences
    • Right-click the 'Mouse Fix' pane
    • Select 'Remove "Mouse Fix" Preference Pane'
  4. Install Mac Mouse Fix again
