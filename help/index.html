Mac Mouse Fix

Notarization Issues

If you're using macOS 10.15 Catalina, you might have encountered the following error message when trying to install Mac Mouse Fix.

Why does this issue occur?

There is a new security feature in Catalina called Notarization. Notarized apps are validated by Apple, so you can be sure they don't contain any malware. Notarization is a great safety measure in principle, but there are some issues in the way it is currently implemented which might prevent you from using Mac Mouse Fix entirely as long as Mac Mouse Fix is not notarized. Notarizing an app costs 100 dollars per year, which unfortunately makes it a nonsolution for free and open-source software like Mac Mouse Fix.
Apple will likely fix these issues in the future, but for now, you will, unfortunately, have to rely on a workaround to get Mac Mouse Fix to run on Catalina.

What is the solution?

Thankfully, Morgan Newman, who has been wonderfully supportive of this little project, figured out how to install Mac Mouse Fix despite this bug. His workaround works by disabling Gatekeeper, a security feature in macOS which prevents the execution of apps that are not signed and notarized with Apple.
To get Mac Mouse Fix to work, you could either disable Gatekeeper entirely, or you could create a Gatekeeper exception specifically for Mac Mouse Fix. Disabling Gatekeeper entirely might make it more likely for you to run malware on accident, but it will prevent any future headache trying to run un-notarized software. Adding an exception will not compromise the security of your system in any way, but it is slightly more complicated, due to some bugs in Catalina.

Solution 1 – Disabling Gatekeeper entirely

  1. Open Terminal and enter the following command

    2. sudo spctl --master-disable

  2. Close System Preferences if it is currently running

    3. Screenshot of Terminal after entering the command

Solution 2 – Adding a Gatekeeper exception for Mac Mouse Fix

  1. Install Mac Mouse Fix
  2. Open Terminal and enter the following command

    3. sudo spctl --add ~/Library/PreferencePanes/Mouse\ Fix.prefPane; sudo spctl --add /Library/PreferencePanes/Mouse\ Fix.prefPane

  3. Uninstall Mac Mouse Fix
    • Right-click its pane in System Preferences and select 'Remove "Mouse Fix" Preference Pane'
  4. Install Mac Mouse Fix again
